## Key Conventions

### Imports
- Import from `@storybook/blocks` for documentation components
- Import stories module for linking to Canvas examples

### Sections
- **Usage**: Quick code example showing basic usage
- **Examples**: Canvas blocks linked to actual stories
- **Props**: Auto-generated props table from Controls
- **Guidelines**: When to use, when not to, accessibility

### Managed Marker
- First line: `{/* @storyteller-managed */}`
- Everything above `CUSTOM DOCS` line is managed
- Everything below is preserved during sync
